# 尘甲系列服务器文档站

尘甲系列服务器的文档站点。

站点基于 [VuePress](https://vuepress.vuejs.org/) 和 [vuepress-theme-plume](https://theme-plume.vuejs.press/) 构建。

## 安装

```sh
npm install
```

## 使用

```sh
# 启动开发服务
npm run docs:dev

# 构建生产版本
npm run docs:build

# 本地预览生产版本
npm run docs:preview

# 更新 VuePress 与主题依赖
npm run vp-update
```
