# 矿石

## **明镜矿石**

只能用镐子类工具挖掘

挖掘后可掉落1个【明镜石】

生成于末地Y值30-70

## **熔火之心**

挖掘后可掉落【七美德-谦逊残片】，使用【七美德-谦逊残片】可兑换神器【七美德谦逊】

生成于下界5-25层
